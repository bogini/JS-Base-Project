JS-Base-Project
======

Base project with Mocha, Chai, Babel7, ES6 and ESLint

### How to run
After cloning the directory, install dependencies with:
```bash
yarn install
```

To run unit tests:
```bash
yarn test:watch
```

To start:
```bash
yarn start
```

To build:
```bash
yarn build
```

To serve:
```bash
yarn serve
```
